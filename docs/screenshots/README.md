# README Screenshots

Drop sanitized product screenshots in this folder before publishing the repository.

Recommended filenames:

- `overview.png`
- `task-editor.png`
- `completed-log.png`

Guidelines:

- Use demo or synthetic tasks only.
- Keep the UI clean and readable at GitHub README size.
- Prefer PNG exports at 2x scale.
- Update the root `README.md` if you choose different filenames.
